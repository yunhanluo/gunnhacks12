# Our Frontend
This is where the frontend for our code lives. It's powered by flutter, meaning that we can deploy it for literally everything, desktop (macOS, Windows, and Linux), mobile (iOS and Android), and even web! 

We didn't want our code to just be a ChatGPT Slop Wrapper, so our bot instead of responding with a big text block organizes nice details into different sections, making it easy to understand everything AI outputs.
